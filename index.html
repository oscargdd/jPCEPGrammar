PCEP Tutorial and Grammar

PCEP stands for Path Computation Element Protocol. Since it initial specification in RFC 5440, a lot of new RFCs and drafts have been published. This web page, powered by a javascript library, aims at compiling the grammar of all PCEP RFCs and drafts. The document draft-cmfg-pce-pcep-grammar-02 expresses the concern with current issues with existing RBNF notation for PCEP messages and extensions.
